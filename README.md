## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Add your profile photo:
   - Place your photo in the `public` folder as `profile.jpg`

3. Update your information:
   - Open `src/components/Hero.jsx` and replace social media usernames
   - Update your name and title
   - Customize the About, Skills, and Projects sections

4. Run the development server:
```bash
npm run dev
```

5. Build for production:
```bash
npm run build
```

## Customization

### Social Media Links
Update the URLs in `src/components/Hero.jsx` and `src/components/Footer.jsx`:
- GitHub: Replace `YOUR_USERNAME` with your GitHub username
- Telegram: Replace `YOUR_USERNAME` with your Telegram username
- TikTok: Replace `YOUR_USERNAME` with your TikTok username
- YouTube: Replace `YOUR_USERNAME` with your YouTube channel name
- Instagram: Replace `YOUR_USERNAME` with your Instagram username
